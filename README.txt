# PsychHelp Enhanced Website

PsychHelp is a modern, responsive mental health and therapy landing page designed and developed by **Rami Almohamad**.
The website presents psychological support services in a clean, warm, and professional style with smooth animations, mobile-friendly navigation, and an attractive user experience.

## Live Demo

[View Website](https://ramialmoh12.github.io/PsychHelp/)

## Project Overview

This project was created to provide a professional online presence for a psychology and therapy service.
The design focuses on trust, comfort, accessibility, and clear communication with visitors.

The website includes service sections, therapy methods, pricing cards, testimonials, blog previews, contact information, and a responsive layout that works smoothly on desktop, tablet, and mobile devices.

## Features

* Modern and warm visual design
* Fully responsive layout
* Sticky glass-style navigation bar
* Mobile burger menu
* Smooth scrolling
* Scroll reveal animations
* Animated statistics counters
* Service and pricing cards
* Testimonials section
* Blog preview section
* Contact and appointment section
* Scroll-to-top button
* Clean and organized code structure

## How to Run the Project

You can run this project directly in the browser.

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation or backend setup is required.

## How to Publish on GitHub Pages

1. Open the GitHub repository.
2. Upload or replace the files:

   * `index.html`
   * `style.css`
   * `app.js`
   * `README.md`
3. Commit the changes.
4. Go to **Settings > Pages**.
5. Choose the main branch and root folder.
6. Save and wait for GitHub Pages to publish the site.
