PsychHelp Enhanced Website

Files included:
- index.html
- style.css
- app.js

How to upload to GitHub Pages:
1. Open your PsychHelp repository on GitHub.
2. Replace your old index.html with this new index.html.
3. Upload style.css and app.js in the same root folder as index.html.
4. Commit the changes.
5. Wait 1-3 minutes and refresh your GitHub Pages link.

Features added:
- Modern warm visual style
- Sticky glass navigation
- Mobile burger menu
- Smooth scrolling
- Reveal animations on scroll
- Animated stats counters
- Hover animations
- Responsive cards and sections
- Scroll-to-top button
